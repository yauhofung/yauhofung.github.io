Python is a relatively popular programming language within the field of Computer Science. However, its popularity does not exempt it from displaying errors when dealing with poorly written codes. Like most other programming languages, there are many different types of errors in Python. Each error usually offers programmers insight on why a certain part of their code is not running correctly.

Common Python Errors

  • NameError
  • OverflowError
  • RuntimeError
  • SyntaxError
  • IndentationError
  • ZeroDivisionError
